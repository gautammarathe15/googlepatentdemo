WHAT EACH STEP DOES
?????????????????????????????????????????????????????????????????????????????????

1. LAUNCH CHROME
   ?? Initializes ChromeDriver with download directory

2. OPEN GOOGLE.COM
   ?? Navigates to https://www.google.com
   ?? Waits 2 seconds for page load

3. ACCEPT COOKIES
   ?? Searches for cookie banner
   ?? Clicks "Accept" button if found
   ?? Continues if not found

4. SEARCH GOOGLE
   ?? Finds search box by name="q"
   ?? Types search term
   ?? Presses Enter key
   ?? Waits 3 seconds for results

5. VERIFY SEARCH RESULT
   ?? Waits for link to patents.google
   ?? Checks element is displayed
   ?? Falls back to page source check

6. NAVIGATE TO PATENTS
   ?? Opens https://patents.google.com directly
   ?? Waits 2 seconds

7. SEARCH PATENTS
   ?? Finds search input on Patents page
   ?? Types search term
   ?? Presses Enter
   ?? Waits 3 seconds for results

8. VERIFY RESULTS
   ?? Checks for result containers
   ?? Multiple fallback strategies
   ?? Last resort: page source validation

9. OPEN FIRST PATENT
   ?? Clicks first result link
   ?? Multiple selector strategies
   ?? Waits 3 seconds for page load

10. TRIGGER PDF DOWNLOAD
    ?? Searches for download button
    ?? Clicks if found
    ?? Continues if not found
    ?? Tracks state in context

11. VERIFY PDF EXISTS
    ?? Checks download directory
    ?? Asserts PDF file present

12. VERIFY DOWNLOAD COMPLETE
    ?? Polls directory every 500ms
    ?? Checks for completed PDF (no temp files)
    ?? Timeout: 60 seconds
